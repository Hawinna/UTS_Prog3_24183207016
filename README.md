## Aplikasi Pemesanan Makanan (Kasir)
Aplikasi kasir sederhana berbasis desktop yang dibangun menggunakan Java Swing (JFrame). Aplikasi ini memungkinkan pengguna untuk mencatat pesanan makanan dan minuman, menentukan jumlah porsi, serta mencetak struk pesanan dalam bentuk dialog konfirmasi.

### Fitur Utama
- Input Data: Mencatat nomor meja dan nama pemesan.
- Kategori Dinamis: Daftar menu otomatis berubah berdasarkan kategori Makanan atau Minuman.
- Manajemen Order: Menambah item dengan jumlah porsi tertentu ke dalam daftar pesanan.
- Cetak Struk: Menampilkan ringkasan seluruh pesanan dalam bentuk dialog struk.
- Reset: Menghapus seluruh data pesanan dan input teks sekaligus.

### Cara Menjalankan
- Buka file kasir.java.
- Klik kanan lalu pilih Run File atau tekan F6.
- Isi No. Meja dan Nama Pemesan pada kolom yang tersedia.
- Pilih kategori pada dropdown Menu (Makanan/Minuman).
- Klik item yang diinginkan pada daftar menu di bawahnya.
- Klik tombol Tambah, lalu masukkan jumlah porsi pada jendela yang muncul.
- Ulangi langkah 2-4 untuk menambah pesanan lain.
- Klik tombol Proses untuk melihat struk belanja akhir.
- Gunakan tombol Hapus jika ingin membatalkan semua pesanan dan memulai dari awal.
